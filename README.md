# ALMOST GONE
Minimal monochrome fashion e-commerce site. Responsive layout with grayscale-to-color hover effects and modern streetwear aesthetic.

## Features

- Responsive design
- Minimal black & white aesthetic
- Grayscale to color hover effect on images
- Hero section
- Product grid
- Categories section
- Journal/blog section
- Customer reviews
- Newsletter block
- GitHub Pages ready

## Technologies

- HTML5
- CSS3
- CSS Grid
- Flexbox

## Preview

Static storefront concept for a fashion brand.

## Deployment

### GitHub Pages

1. Create a new GitHub repository
2. Upload `index.html`
3. Open repository **Settings**
4. Open **Pages**
5. Select:
   - Deploy from branch
   - `main`
   - `/root`
6. Save

The site will be published automatically.

## Notes

- Buttons are currently non-functional
- Images are loaded from Unsplash
- No backend or cart logic included yet
- Can be expanded into a full online store
